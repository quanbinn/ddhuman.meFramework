# Creating an app

1. 
```bash
$ mkdir ddhumanMe && cd ddhumanMe
```

2.
```bash
$ npm init egg --type=simple
```

3. 
```bash
$ npm install
```

4. 
```bash
$ npm run dev
```

5. 在浏览器里输入： http://localhost:7001
